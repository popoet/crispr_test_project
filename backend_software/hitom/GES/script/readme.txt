执行本脚本前需配置好参数:

BASE_DIR	为当前的运行路径

fq_path	为输入数据的路径

upload_data	为客户指定目标序列的文件的绝对路径，其中文件一共3列，\t 分隔 。第1列'SamID' : 样本名称 ; 第2列'amplicon_seq' : 参考序列; 第3列'guide_seq': 目标序列; 

thread	为执行的线程数目

memory	为分配的内存大小

start	设置为 0 时，输出的就是目标序列比对上参考序列的起始位置 ，设置为 a (>0) 时，输出的就是目标序列比对上参考序列的起始位置-a 的位置 ；
end	设置为 0 时，输出的就是目标序列比对上参考序列的终止位置 , 设置为 b (>0) 时，输出的就是目标序列比对上参考序列的终止位置+b 的位置 ；


output.tsv	结果文件，其中该文件一共6列，\t分隔。第1列:'变异编号'， 第2列:'支持reads数目'，第3列:'变异比例',第4列:'变异类型'，第5列:'变异序列'，第6列:'扩增片段序列';
		其中第4列，WT 表示野生型（未发生变异）；I 表示碱基插入；D 表示碱基缺失；SNP 表示碱基替换；第6列，小写字母 表示插入或替换后的碱基，- 表示缺失的碱基；
